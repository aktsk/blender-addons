# undo_without_camera

## setup

```
$ /Applications/Blender.app/Contents/MacOS/Blender
```

and 

In `Prefarence > Fole Paths > Data > Scripts`, set `/path-to-repo/scripts/`.

## dev

```
$ /Applications/Blender.app/Contents/MacOS/Blender
```

and

1. edit codes.
2. In `Prefarence > Addons > Testing`, disable and enable `Object: undoWithoutCamera`.
